## 깃(git)

__깃은 개발자가 코드를 저장하고 관리할 수 있는 버전 관리 시스템__

여러 명의 사용자들 간에 해당 파일작업을 조율하기 위한 버전 관리 시스템으로
파일의 변경사항을 추적하기 위하여 파일수정시 메세지 작성, 저장관리가 매우 중요하다.
